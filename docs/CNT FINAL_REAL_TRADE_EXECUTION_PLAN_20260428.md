## 📊 반드시 확인해야 할 6가지 (핵심)

### 1. 주문 생성
```text
EXPECTED: 주문 1회 발생
FAIL: 주문 없음 → 신호/필터 문제
```

### 2. 체결 여부
```text
EXPECTED: filled 또는 partial
FAIL: 미체결 → execution 문제
```

### 3. 포지션 생성
```text
EXPECTED: open_trade != null
FAIL: 없음 → 주문 실패
```

### 4. 청산 정상 여부
```text
EXPECTED: TP/SL 정상 종료
FAIL: 고착 → 위험
```

### 5. 로그 사이클
```text
runtime.log → portfolio.log → snapshot
모두 이어져야 정상
```

### 6. LIVE_GATE 유지
```text
EXPECTED: LIVE_READY 유지
FAIL: 즉시 문제
```
